# Diagrams

This directory will store diagrams for the AWS EC2 Hardening & Vulnerability Management project.

## Planned Diagrams

Potential diagrams may include:

- EC2 workload placement diagram
- SSH access flow diagram
- Security group exposure diagram
- Patch management workflow
- Vulnerability assessment workflow
- Systems Manager access flow

## Diagram Requirements

Diagrams should clearly show:

- EC2 instance
- VPC/subnet placement
- Security group
- Administrative access path
- Optional Systems Manager access path
- Monitoring or assessment flow

## Security Notes

Diagrams should not expose sensitive information.

Do not include:

- AWS account ID
- Access keys
- Secret keys
- Private SSH keys
- Personal email addresses
- Billing information
- Unredacted public IPs unless intentionally documented
- Sensitive resource identifiers
