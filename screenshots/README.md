# Screenshots

This directory will store redacted screenshots for the AWS EC2 Hardening & Vulnerability Management project.

## Screenshot Rules

Before uploading screenshots, verify that they do not expose sensitive information.

Do not commit screenshots showing:

- AWS account ID
- Access keys
- Secret access keys
- Private SSH keys
- Billing details
- Personal email addresses
- Unredacted ARNs if sensitive
- Public IP addresses unless intentionally documented
- Private information from the AWS console

## Planned Screenshot Evidence

Potential screenshots may include:

- EC2 instance configuration
- Security group rules
- IAM role attachment
- SSH access validation
- OS update output
- Systems Manager managed instance status
- Amazon Inspector finding, if used
- CloudTrail EC2 event

## Redaction Requirement

All screenshots must be reviewed and redacted before being committed.
